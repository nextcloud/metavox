**Document voor onderhandelingen met Nextcloud GmbH**

## 1. Executive Summary

MetaVox is een strategisch waardevolle Nextcloud app die semantische metadata toevoegt aan documenten. Met 1.000+ downloads en een 6-sterren rating is de app production-ready en wordt deze overwogen voor integratie in Nextcloud Core.

**Aanbevolen vergoeding: €100.000 - €150.000**

De waarde wordt primair bepaald door MetaVox's rol als **SharePoint migratie enabler** - precies de markt waar Nextcloud op inzet met EU-overheden.

## 3. Strategische Meerwaarde voor Nextcloud

### 3.1 SharePoint Migratie Enabler (KRITISCH)

Dit is de **belangrijkste strategische waarde** van MetaVox.

#### Context

- EU-overheden (o.a. Schleswig-Holstein, Duitsland) hebben aangekondigd SharePoint te vervangen met Nextcloud
- Stad Keulen werkt actief met Nextcloud aan SharePoint-alternatieven
- Migraties betreffen vaak **honderden miljoenen documenten** met metadata

#### Probleem zonder MetaVox

SharePoint documenten hebben uitgebreide metadata (auteur, afdeling, classificatie, retentiedata, etc.). Zonder MetaVox gaat deze metadata verloren bij migratie, wat:

- Compliance-problemen veroorzaakt (GDPR, archiefwet)
- Document retrieval bemoeilijkt
- Bedrijfsprocessen verstoort

#### Oplossing met MetaVox

- API ondersteunt bulk-import van metadata velden
- Gestructureerde mapping van SharePoint metadata naar Nextcloud
- Behoud van semantische context bij migratie

#### Bewijs van Waarde

LinkedIn post Rik Dekker (20 september 2025):

> "MetaVox is on fire! 🔥 [nieuwe API release] enables: automatically populating metadata fields, creating fields directly through the API, supporting advanced migration scenarios. Use case: migrating SharePoint files with metadata into Nextcloud."

### 3.2 Enterprise-Ready Functionaliteit

| Feature | Belang voor Enterprise |
|---------|----------------------|
| Role-based metadata access | Voldoet aan enterprise security requirements |
| Workflow Engine integratie | Automatisering van document classificatie |
| JSON import/export | Interoperabiliteit met andere systemen |
| Team folder scoping | Past in enterprise folder structuren |

### 3.3 Target Markten

MetaVox bedient precies de markten waar Nextcloud groeit:

- **Overheid** - Digitale soevereiniteit, compliance
- **Onderwijs** - Universiteiten, hogescholen (ontwikkelingscontext)
- **Gezondheidszorg** - Documentclassificatie, retentie
- **Juridisch** - Case management, archieven

### 3.4 Concurrentievoordeel vs Microsoft

Met MetaVox in core kan Nextcloud claimen:

- ✅ Volledige SharePoint metadata migratie support
- ✅ On-premise, GDPR-compliant alternatief
- ✅ Geen vendor lock-in

## 6. Onderhandelingskader

### 6.1 Onderhandelingspunten

#### Minimaal Acceptabel

- €75.000 eenmalig
- Credit in Nextcloud documentatie
- Co-marketing bij SharePoint migratie messaging

#### Ideaal Pakket

- €100.000+ eenmalig
- 1-jaar paid maintenance/development contract
- Named credit als originele ontwikkelaars
- Mogelijkheid om als Nextcloud partner verder te werken (FormVox, IntraVox)

#### Niet-Financiële Waarde

- Nextcloud "certified partner" status
- Priority app store placement voor andere apps (FormVox, IntraVox)
- Speaking slot bij Nextcloud Conference
- Joint press release over integratie

### 6.2 Onderhandelingstips

#### Do's

1. **Benadruk SharePoint migratie waarde** - Dit is het sterkste argument
2. **Toon traction** - 1000+ downloads, 6★ rating, actieve gebruikers
3. **Refereer naar enterprise klanten** - UvA, HvA als proof points
4. **Positioneer als partner** - FormVox en IntraVox als portfolio
5. **Wees flexibel in structuur** - Cash vs. contract vs. revenue share

#### Don'ts

1. **Onderschat jullie positie niet** - Nextcloud heeft dit nodig voor SharePoint migraties
2. **Accepteer niet alleen "exposure"** - Dit is significante waarde, niet een hobby project
3. **Vergeet IP overdracht niet** - AGPLv3 is al open, maar trademark/branding heeft waarde

## 8. Conclusie

MetaVox vertegenwoordigt een strategische investering voor Nextcloud met een waarde die significant hoger ligt dan de pure ontwikkelkosten. De app is:

1. **Production-ready** - 11 releases, 6★ rating, actieve gebruikers
2. **Strategisch essentieel** - Enablet SharePoint migraties voor EU-overheden
3. **Enterprise-grade** - Role-based access, Workflow integratie, API support
4. **Uniek** - Geen vergelijkbare oplossing in het Nextcloud ecosystem

Een vergoeding van **€100.000 - €150.000** is redelijk en marktconform gezien:

- De ontwikkelinvestering (~1000 uur senior development)
- De strategische waarde voor Nextcloud's groeimarkt
- De volwassenheid en production-readiness van de app
